Here, we will be walking through some elementary probability theory, mostly aimed at use in Statistical Mechanics - which is an inherently probabilistic description of physical systems.

General Definitions

The main object of investigation in probability is the random variable.

Definition: Random Variable

A random variable is an object that assumes values from a defined set of outcomes. The set of such outcomes may be discrete or continuous.

Every experiment that we seek to investigate using probability has a sample space, which is simply a set that contains all the possible outcomes that might happen in a particular run of the experiment. A random variable is thus a map from the sample space to another set.
